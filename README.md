# BottomTopDialog
将dialog显示在底部和顶部

参考的[Android-AlertView](https://github.com/saiwu-bigkoo/Android-AlertView)

![screenshot](/screenshot.gif)

### 使用

``````
allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
  
 dependencies {
	        compile 'com.github.jeffreyhappy:BottomTopDialog:v0.1'
	}
``````
